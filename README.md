BEM
  BEM là tiêu chuẩn đặt tên class khi viết CSS
  Ý nghĩa: là viết tắt của Block Element modifier
  Block: Khối
  Element: Thành phần trong khối
  Modifier: Bổ sung ý nghĩa cho block hoặc element

  Cú pháp:
  .block
  .block__element
  .block--modifier
  .block__element--modifier
